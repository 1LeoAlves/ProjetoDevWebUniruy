## Instalação

Instale e extraia os arquivos. 
Baixe o XAMPP ou WAMPP e mova a pasta "webcars" para dentro da pasta de hospedagem.
Inicie o XAMPP/WAMPP, e abra o "localhost/webcars/alugar-carros-main/home.php".
Pronto!
